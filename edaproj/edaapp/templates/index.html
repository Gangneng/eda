1. 분석 목적과 변수 확인




분석 목적

심장마비 가능성이 높은 사람들의 특징이 무엇일까?


변수

  • age : 나이
  • sex : 성별 남성이면 0, 여성이면 1
  • cp : 가슴 통증 종류
    • 1 : 전형적 협심증
    • 2 : 비정형 협심증
    • 3 : 비 협심증
    • 4 : 무증상
  • trtpbs : 혈압
  • chol : BMI 센서를 통해 가져온 콜레스테롤
  • fbs : 공복 혈당이 정상이면 0, 120 이상이면 1
  • restecg : 휴식 심전도 결과
    • 0 : 정상
    • 1 : ST-T파 이상
    • 2 : 좌심실 비대 가능성을 보이거나 확실한 경우
  • thalachh : 최대 심박수 달성
  • exng : 운동으로 인한 협심증
  • oldpeak : 이전 피크
  • output : 심장마비 가능성 적으면 0, 많으면 1

3. 가설 및 검증 결과




1. 나이가 많으면 심장마비 가능성이 높을 것이다.



{% load static %}
40-50대에서 심장마비 가능성이 가장 많다.



2. 심장마비 가능성 여부는 성별과 상관관계가 있다.



{% load static %}
남성이 여성에 비해 심장마비 확률이 압도적으로 높았다.



3. 가슴 통증의 종류와 심장마비 가능성 여부는 상관관계가 있다.



{% load static %}
비정형 협심증, 비협심증 환자는 심장마비 확률이 더 높다.



4. 최대 심박수가 높으면 심장 마비 가능성이 높을 것이다.



{% load static %}
최대 심박수가 140 이상으로 넘어가면 심장 마비 가능성이 더 높아진다.



5. 나이가 많으면 많아질수록 콜레스테롤이 높아질 것이고 그에 따라 심장 마비 가능성이 높아질 것이다.



{% load static %}
나이와 콜레스테롤은 큰 관계가 없고, 콜레스테롤 자체도 심장 마비와는 큰 관련성을 나타내고 있지 않다.
